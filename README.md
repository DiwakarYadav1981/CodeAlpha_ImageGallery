# CodeAlpha_ImageGallery

A responsive image gallery with lightbox view and category filters, built with HTML, CSS, and JavaScript for the CodeAlpha Frontend Development Internship.

## Features
- Grid-based responsive layout
- Lightbox view with next/prev navigation (click, buttons, or arrow keys)
- Category filters: Nature, City, People, Animals
- Hover effects with title overlay and smooth image zoom
- Fade-in animation on load
- Keyboard support in lightbox (Esc to close, arrows to navigate)

## Tech Stack
- HTML5
- CSS3 (Grid, animations)
- Vanilla JavaScript

## How to Run
1. Clone this repository
2. Open `index.html` in any browser
3. Click any image to open it in the lightbox

## Note on Images
Images are pulled from placeholder URLs (picsum.photos) for demo purposes.
